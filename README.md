# website

This is a simple website for [Web Scrobbler][web-scrobbler-github] extension.

## Development

```sh
# Install dependencies
> npm install

# Lint project files
> npm run lint

# Format project files
> npm run format

# Run a server with live reload on changes
> npm start
```
