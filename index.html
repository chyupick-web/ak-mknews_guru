Web Scrobbler
Scrobble music all around the web!

Web Scrobbler helps online music listeners to scrobble their playback history.

Key features

  • Support for Last.fm, Libre.fm, ListenBrainz.
  • Now playing notifications.
  • Built-in track metadata filter.
  • A lot of supported websites.

Supported browsers

  • Chrome and Chrome-based browsers (Opera, Vivaldi etc.)
  • Firefox
